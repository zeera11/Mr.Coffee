# MR.COFFEE

MR.COFFEE is a responsive coffee shop website designed to bring the warmth, creativity, and community spirit of the coffee shop into the digital world. Users can explore menus, discover rewards, learn about the company, and place orders through an intuitive web experience.


## Figma Prototype

https://www.figma.com/design/73BcYaEB8tg9ukih93KOsw/Mr.-Coffee--project-lab-?node-id=0-1&t=gQh5jOKFOLJ4RbmX-1


## Features

- Home Page with featured products and promotions
- Rewards Page with app download and promo sections
- About Us Page showcasing company story, mission, and vision
- Menu Page with category-based product filtering
- Order Page with interactive form validation
- Responsive design for desktop, tablet, and mobile devices

## Tech Stack

- Figma
- HTML5
- CSS3
- JavaScript

## Responsive Design

The website is optimized for:

- Desktop
- Tablet
- Mobile

using Media Queries and Viewport Configuration.
